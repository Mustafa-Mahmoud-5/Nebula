# Nebula is a simple nodejs expressjs mini-framework with handy built-in functions and organized design pattern for starting a server

## Plugins:
Nodejs, ExpressJs, mongoose, jwt, express-validator, Cloudinary, multer

## Structure:

    ├───Apis
        ├───auth.js
    ├───helpers
        ├───cloudinary.js
        ├───expiryDates.js
        ├───pagination.js
        ├───randomIds.js
        ├───sendError.js
    ├───images
        ├───preventer.js
    ├───middlewares
        ├───checkValidation.js
        ├───isAuth.js
        ├───multerUploader.js
    ├───models
        ├───User.js
    └───routes
        ├───auth.js
    ├───app.js
    ├───.env
